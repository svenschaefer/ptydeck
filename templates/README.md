# Repository Templates

This directory holds reusable delivery templates for recurring repository work.
They are intentionally lightweight and deterministic rather than full project generators.

Current templates:

- `templates/backend-endpoint/`
  - snippets and checklists for adding a backend HTTP endpoint in the current `runtime.js` + OpenAPI layout
- `templates/frontend-ui-module/`
  - controller/test templates for new frontend UI modules following the current factory-style controller pattern

Use the templates as repository-local starting points.
They are not a promise that every new feature should be generated mechanically.
